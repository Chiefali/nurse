*A smart, data-driven platform connecting families with qualified Community Health Workers in Atlanta to reduce loneliness and provide on-demand care.*

---

## 🚀 Latest Features

- Real-time moving nurse/CHW location tracking on map
- Improved mobile responsiveness
- Multi-language support (English / Español)
- Voice Assistant + Dial-in simulation
- Live Chat with CHW Support
- Stripe Payment Mock

---

## ✨ Key Features

### For Families
- Browse nearby CHWs with ratings and specialties
- **Real-time GPS tracking** of available workers
- Voice & Dial-in assistant
- Instant or scheduled booking with mock payment
- Multi-language support

### For Community Health Workers
- Professional profile & availability management
- Job notifications
- Easy booking acceptance

### Analytics
- Live demand heatmap using Census + CDC data
- Intelligent matching system
